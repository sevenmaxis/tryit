# Tryit

another version of try

Instead of some_object.try(:gsub!, /\D/, '') you can do this some_object.tryit{ gsub!(/\D/, '') }

## Installation

Add this line to your application's Gemfile:

    gem 'tryit'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tryit
